# 실시간 ROM 교육용 피드백 앱 - Browser-only v3

이 버전은 Streamlit 서버에서 MediaPipe/OpenCV를 실행하지 않고, 사용자의 브라우저 안에서 카메라와 MediaPipe Web을 실행합니다.

## v2 개선 내용

- 스마트폰에서 음성이 나오지 않는 문제를 줄이기 위해 **음성 활성화/테스트** 버튼 추가
- 측정 중 카메라는 유지한 채 분석만 멈추는 **측정 중단** 버튼 추가
- **측정 재개**, **측정 새로 시작**, **카메라 정지** 버튼을 사이드바와 영상 아래에 모두 배치
- 측정 새로 시작 버튼이 측정 중에도 작동하도록 상태 초기화 로직 개선
- 스마트폰에서 음성 안내가 차단될 때 화면 안내 메시지 표시

## 스마트폰 사용 안내

- 카카오톡/메신저 내부 브라우저보다 Chrome, Samsung Internet, Safari 같은 일반 브라우저 권장
- HTTPS 주소로 접속
- 카메라 권한 허용
- 음성이 나오지 않으면 먼저 **음성 활성화/테스트** 버튼을 누르고, 스마트폰 무음모드/미디어 볼륨을 확인

## 주의

단일 웹캠 기반 교육/피드백용 프로토타입입니다. 임상 진단 또는 치료 결정에 사용하려면 별도 타당도 검증이 필요합니다.


## v3 개선 내용

- 고정 십자선만 표시하던 방식에서, 신체 기준점에 맞춘 동적 기준선을 추가했습니다.
- 정면 촬영: 코/어깨 중심선, 어깨 수평선, 필요 시 골반 기준선을 표시합니다.
- 측면 촬영: 귀/코-어깨 또는 어깨-골반 수직 정렬선을 표시합니다.
- 음성 안내를 단계별로 명확화했습니다: 준비자세 유지 → 움직임 시작 신호 → 끝범위 후 복귀 신호 → 1회 완료/다음 회 안내.
- 반대 방향 움직임 또는 기준선 불일치 복귀는 기록하지 않고 재정렬 단계로 돌립니다.


## v4 변경 사항

- 음성 안내 문구를 교육용 ROM 측정 절차에 맞게 수정했습니다.
- 움직임 시작 직후 바로 복귀를 안내하지 않고, 선택한 방향으로 끝범위까지 움직인 뒤 끝범위에서 잠시 멈추도록 안내합니다.
- 끝범위에서 약 0.8초 안정적으로 유지되면 “끝범위 움직임이 인식되었습니다. 이제 처음 위치로 천천히 돌아오세요.”라고 안내합니다.
- 끝범위에서 충분히 멈추지 않고 바로 돌아오면 해당 반복을 기록하지 않도록 했습니다.
- 정상 측정 후 다음 회차 안내는 “준비자세로 돌아와 기준선을 다시 맞춰 주세요.”로 통일했습니다.


## v5 수정 사항

- 측정 동작 드롭다운에서 다른 동작을 선택해도 첫 번째 항목으로 되돌아가던 문제를 수정했습니다.
- 측정 동작 변경 시 상단 안내문과 측정 알고리즘이 즉시 함께 바뀌도록 수정했습니다.
- 측정 중 동작을 바꾸면 기존 측정을 초기화하고 새 동작의 준비자세부터 다시 시작합니다.
- 동작은 자동으로 추정되는 것이 아니라, 사용자가 선택한 동작과 촬영 방향에 맞춰 측정됩니다.

## v7 개선 사항

- 끝범위 정지 시간을 기본 1.5초로 늘리고, 사용자가 0.8~3.0초 범위에서 조절할 수 있도록 했습니다.
- 화면 중앙에 고정 수직/수평 기준축과 허용 범위 밴드를 표시합니다.
- 준비자세 판정은 주요 마커가 고정 기준축 허용 범위 안에 들어오는지로 확인합니다.
- 목 정면 측정은 코와 어깨 중심, 목 측면 측정은 귀/코와 어깨 중심, 몸통은 어깨/골반 중심, 사지 관절은 중심 관절이 기준축 주변에 있는지를 기준으로 합니다.
- 선택 방향 움직임이 0.3초 이상 유지되어야 움직임 시작으로 인정합니다.
- 끝범위에서 충분히 멈춘 뒤에만 “처음 위치로 돌아오세요” 안내가 나오도록 했습니다.
- 처음 위치 복귀도 0.6초 이상 안정되어야 1회 측정으로 기록합니다.
- ROM 값을 5프레임 이동평균으로 안정화하여 갑작스러운 landmark 흔들림으로 인한 과대값을 줄였습니다.


## v7 음성 안내 개선

- 스마트폰 브라우저에서 Web Speech API가 차단되거나 `onend` 이벤트가 오지 않아 다음 안내가 멈추는 문제를 줄였습니다.
- `음성 활성화/테스트` 버튼을 누르면 음성합성과 함께 Web Audio 신호음을 같이 해제합니다.
- 음성 출력이 실패해도 단계 변화가 전달되도록 신호음과 진동을 보조 안내로 제공합니다.
- Android/Safari에서 음성 큐가 멈추지 않도록 watchdog 타이머를 추가했습니다.

스마트폰에서 음성이 들리지 않으면 다음을 확인하세요.

1. 카카오톡/메신저 내부 브라우저가 아니라 Chrome, Samsung Internet, Safari에서 직접 열기
2. 무음모드 해제 및 미디어 볼륨 올리기
3. 앱 화면에서 `음성 활성화/테스트` 버튼을 직접 한 번 누르기
4. 음성은 안 들려도 신호음/진동과 화면 문구가 나오면 브라우저 음성합성 차단 가능성이 큼
